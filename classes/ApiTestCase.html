\ApiTestCase

ApiTestCase is the base class for testing all RESTful API controller methods.

Usage:

$response = $this->post(
     'api/login',
     array(
         'username'=>'userName',
         'password'=>'password'
     ),
     array(
         CURLOPT_CRLF=>true,
         'cookies'=>array(
             'giud'=>'value',
             ...
         )
     )
);
